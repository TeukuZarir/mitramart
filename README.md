# 🛒 Mitramart Enterprise - POS & Inventory System

Sistem Point of Sales (POS) dan Manajemen Inventaris terpadu untuk operasional Mitramart. Repositori ini berisi *source code* lengkap untuk pengembangan dan proses *deployment*.

##  Tech Stack Utama
* **Environment:** Node.js
* **Backend:** NestJS *(atau sesuaikan jika menggunakan framework lain)*
* **Database & Auth:** Supabase

##  Cara Menjalankan Secara Lokal (Run Locally)

**Prerequisites:** Pastikan [Node.js](https://nodejs.org/) sudah terinstall di komputermu.

**Langkah-langkah instalasi:**

1. **Install dependencies:**
   Buka terminal di direktori proyek ini, lalu jalankan perintah:
   ```bash
   npm install
